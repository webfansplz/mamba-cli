# 快捷搜索

> 通过默认浏览器快捷搜索, 默认使用Google

```shell
$ null search <keyword>
```

## 参数

| name |       alias     |          describe       |
| :--: | :-------------: | :---------------------: |
|  -n  |      --npm      |      search by npm      |
|  -s  | --stackoverflow | search by stackoverflow |
|  -w  |      --wiki     |      search by wiki     |
|  -b  |      --baidu    |      search by baidu    |
|  -g  |     --github    |      search by github   |
